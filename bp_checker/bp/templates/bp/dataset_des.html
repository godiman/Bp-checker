Blood Pressure Dataset

Data set description.

The data set used in this project was gotten from Kaggle.com. It contains the following attributes:

Education: The values are 1 (primary) , 2 (secondary), 3 (graduate) , 4 (master)
Age: The value of the age is integer
Body Mass Index: It is the division of weight by height.
Curent Smoker: The value of current smoker is 0 (No) and 1 (Yes)
Heart Rate: The current heart rate.
